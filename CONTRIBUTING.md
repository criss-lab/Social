# Contributing\n\nThank you for your interest in contributing to Bluesky! Here's how to get started:\n\n## Development Setup\n\n1. Fork and clone the repository\n2. Install dependencies: `pnpm install`\n3. Create a feature branch: `git checkout -b feature/my-feature`\n4. Make your changes\n5. Run tests: `pnpm test`\n6. Commit with clear messages\n7. Push and create a pull request\n\n## Code Standards\n\n- Use TypeScript\n- Follow ESLint configuration\n- Add tests for new features\n- Keep commits atomic and descriptive\n- Update documentation as needed\n\n## Running Tests\n\n```bash\n# All tests\npnpm test\n\n# Specific package\npnpm --filter=@bluesky/atprotocol test\n\n# Watch mode\npnpm test -- --watch\n```\n\n## Linting\n\n```bash\npnpm lint\n```\n\n## Type Checking\n\n```bash\npnpm type-check\n```
